## User Story

- as a user, I want to **search for the recipes**, so that i can find new ideas for the meals
- as a user, I want to update to update the nubr of servings, so that i can cook a meal for differentnumber of people
- as a user, I want to bookmark recipes, so that i can review them later
- as a user, I want to be able create my own recipes, so that i have them all organized in the same app
- as a user, I want to be able to see my bookmark and own recipes when i leave the app and comeback later, so that i can close the app safely after cooking

## feature

- Search functionality
- Display pagination
- Display recipe
- Change recipes
- Bookmark function
- User Upload own recipes
- User recipes auto bookmark
- User can only see their recipe
- Store bookmark data in local storage
- Display local storage
